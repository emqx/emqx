Added focused `/api-spec` endpoints and a dashboard API spec explorer page for easier browsing of EMQX HTTP API documentation.
The dashboard now serves tag-scoped and drill-down OpenAPI slices, and these endpoints are disabled together with Swagger when `dashboard.swagger_support` is set to `false`.
Added `emqx ctl api_keys` CLI commands to list, show, add, delete, enable, and disable API keys from the command line.
