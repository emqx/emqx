Added support for using AWS IAM Roles Anywhere with Kafka Producer/Consumer Connectors.  The credential process' HTTP API can be configured in those Connectors.

The credential helper process is expected to be running and accessible by EMQX.  See [its official docs](https://docs.aws.amazon.com/rolesanywhere/latest/userguide/credential-helper.html#credential-helper-serve) for more info.
